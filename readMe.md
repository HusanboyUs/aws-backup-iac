# Website Backup Infrastructure

Terraform configuration for provisioning an AWS S3 bucket used to store database backups for my personal website.

## Overview

This project creates and configures the AWS infrastructure required for backup storage, including:

* S3 bucket for database backup files
* Bucket configuration and security settings
* Infrastructure managed as code using Terraform

## Prerequisites

* Terraform
* AWS account
* AWS credentials configured locally

## Usage

Initialize Terraform:

```bash
terraform init
```

Review the execution plan:

```bash
terraform plan
```

Apply the infrastructure:

```bash
terraform apply -auto-approve
```

## Purpose

Database backup files generated by the website are uploaded to the provisioned S3 bucket, providing durable and centralized backup storage.

## Cleanup

To remove all resources:

```bash
terraform destroy
```
